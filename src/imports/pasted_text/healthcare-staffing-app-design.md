Create a premium, modern mobile app visual prototype for a healthcare and dental staffing platform. The app connects workers/locums with healthcare and dental organisations who need temporary staff for shifts/jobs.

The design should look professional, clean, trustworthy, healthcare-focused, and suitable for a UK-based staffing platform. Use a modern SaaS/mobile app style with rounded cards, soft shadows, clean spacing, and a premium colour palette.

Brand:
- Company/developer brand: Electra
- App purpose: healthcare and dental staffing platform
- Style: modern, professional, premium, clean, trustworthy
- Colour direction: deep navy blue, teal/green healthcare accent, white/light grey background
- Font style: modern sans-serif similar to Inter or Poppins
- Target users: workers/locums, organisations, admin team, super admin/business owner, and customers/individual users

Create mobile app screens first. Use iPhone/Android mobile frame size. Also include a few desktop admin dashboard mockups.

Main goal:
Show what the app will look like visually for a client presentation. The prototype should demonstrate the main user journey, not every tiny setting.

Design requirements:
- Use realistic placeholder data
- Use healthcare/dental icons where suitable
- Make the UI feel like a real professional staffing platform
- Use consistent buttons, cards, badges, spacing, and navigation
- Make worker screens mobile-first
- Make admin screens desktop dashboard style
- Keep the design clean and not overcrowded
- Add small details like notification badges, verification badges, profile avatars, and status chips
- Avoid making it look like a generic AI-generated template
- Use strong visual hierarchy and polished spacing
- Include enough screens for a PowerPoint client presentation
